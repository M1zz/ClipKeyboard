## 🅰️ 메인 (계좌번호 / 식은땀 — 가장 솔직한 버전)

```
나는 숫자를 잘 못 외운다. 특히 계좌번호.

거래처가 "입금할게요, 계좌 좀요" 하면 매번 은행 앱 켜서 복사하고, 다시 대화창 돌아와 붙여넣는다. 별거 아닌 것 같은데 이게 하루에도 몇 번씩이다. 한 번은 숫자 하나 틀린 걸 보내기 직전에 발견하고 식은땀 났다.

그게 싫어서 그냥 키보드에 계좌번호를 박아뒀다. 이제 "계좌"만 치면 키보드 위에 떠서 탭 한 번. 정산 문구도, 자주 쓰는 답장도 다 넣어놨다.

별거 아닌데, 이러고 나니까 마음이 좀 가벼워졌다. 하루에 스무 번씩 하던 그 사소한 신경을 안 써도 되니까.

만들 땐 이게 팔릴까 싶었는데, 어차피 내가 매일 쓰려고 만든 거라 후회는 없다.
ClipKeyboard, 앱스토어에 있다 → https://apps.apple.com/kr/app/clip-keyboard-quick-phrases/id1543660502
```

## 🅱️ 대안 (반복 답장 / 자괴감 버전)

```
메일 쓰다가 문득, 이 문장 지난주에도 똑같이 썼다는 걸 깨달을 때가 있다.
"확인했습니다, 검토 후 회신드리겠습니다." — 이걸 도대체 몇 번이나 다시 쳤을까.

별로 안 힘든 일인데, 같은 걸 매번 처음부터 치는 게 묘하게 사람을 지치게 한다.

그래서 자주 쓰는 답장이랑 정산 문구, 계좌, 내 소개 같은 걸 키보드에 넣어놨다.
이제 짧은 이름만 치면 떠서 탭 한 번. 카톡이든 메일이든 앱 안 나가고.

대단한 건 아닌데, '또 같은 말 치고 있네' 싶은 그 순간이 사라지니까 하루가 조금 덜 피곤하다.

내가 쓰려고 만든 거라 그냥 여기 둔다.
ClipKeyboard → https://apps.apple.com/kr/app/clip-keyboard-quick-phrases/id1543660502
```
