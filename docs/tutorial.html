STEP 01

시작하기 — 키보드 추가 & 전체 접근

iOS 보안 정책상, 사용자가 직접 키보드를 활성화해야 합니다. 1분이면 됩니다.

키보드 추가

  1. iOS 설정 앱 → 일반 → 키보드 → 키보드 → 새로운 키보드 추가…
  2. 목록에서 클립키보드를 선택

전체 접근 허용

  1. 같은 화면에서 방금 추가한 클립키보드를 탭
  2. 전체 접근 허용 토글을 켜기
전체 접근은 키보드가 메인 앱에 저장된 메모에 접근하기 위해 필요합니다. 저장된 데이터는 오직 당신의 기기에만 존재합니다.

키보드 전환

아무 앱에서 키보드를 열고 좌하단 🌐 아이콘을 길게 눌러 클립키보드를 선택하세요.

STEP 02

메모 저장·편집

자주 쓰는 문구를 저장하면 키보드에서 바로 꺼내 쓸 수 있습니다.

메모 추가

  1. 앱 메인 화면에서 우상단 ＋ 버튼 탭
  2. 제목과 내용을 입력 (내용은 실제로 입력될 텍스트)
  3. 필요하면 테마(카테고리)를 선택해 분류
  4. 저장하면 키보드에서 바로 보입니다

키보드에서 사용

  • 저장한 메모 카드를 탭 → 바로 커서 위치에 입력
  • 길게 눌러 복사·편집 메뉴
  • 상단 검색창으로 빠르게 찾기
이모지 · 줄바꿈 · 마크다운은 있는 그대로 입력됩니다. 이메일 서명 같은 긴 텍스트도 문제없습니다.

이미지 메모

사진을 첨부한 메모는 탭하면 클립보드에 이미지로 복사됩니다. 명함·영수증을 저장해두면 유용합니다.

STEP 03

템플릿 & 플레이스홀더

매번 조금씩 달라지는 문구는 {변수}로 남겨두고, 쓸 때만 채우세요.

문법

중괄호 { } 안에 변수 이름을 쓰면 플레이스홀더가 됩니다. 예:

  • 안녕하세요 {이름}님, {날짜}에 예약 확정되었습니다.
  • 주문번호 {order_id} · 배송일 {date}

사용하기

  1. 템플릿 메모를 탭하면 각 플레이스홀더 입력 창이 뜹니다
  2. 값을 채우고 확인하면 완성된 문장이 입력됩니다
  3. 한 번 입력한 값은 자동으로 기록돼 다음에 선택하기 쉽습니다 (예: "이름"에 자주 쓰는 사람 목록)
같은 이름의 변수는 여러 템플릿에서 공유됩니다. {이름}에 입력한 값은 다른 템플릿의 {이름}에서도 제안됩니다.
STEP 04

스마트 클립보드

복사한 텍스트를 자동으로 분류해서, 언제든 다시 꺼내 쓸 수 있게 합니다.

자동 인식하는 15가지 타입

📧 이메일
📞 전화번호
📍 주소
🔗 URL
💳 카드번호
🏦 IBAN / 계좌
🌐 SWIFT / BIC
🧾 VAT / Tax ID
💱 통화·금액
📦 운송장
💵 PayPal / Wise
🔢 숫자
📝 일반 텍스트
🖼️ 이미지
⋯ 기타

어떻게 작동하나요

  • 복사하면 정규식 기반으로 자동 분류 (신뢰도 점수 계산)
  • 최대 100개까지 히스토리 유지
  • 임시 항목은 7일 후 자동 삭제
  • 중요한 항목은 고정(📌)해서 영구 보관
민감 항목(카드번호·계좌·Tax ID)은 마스킹되어 표시됩니다. 탭할 때만 실제 값이 복사됩니다.
STEP 05

콤보 — 여러 메모를 한 번에

이름·이메일·전화번호를 순서대로 자동 입력하는 매크로 같은 기능입니다.

이럴 때 유용

  • 회원가입 폼에 이름·이메일·연락처 일괄 입력
  • 주문서에 배송지·요청사항·할인코드 한 번에
  • CS 답변 = 인사말 + 답변 본문 + 서명 조합

콤보 만들기

  1. 앱 메인 → 콤보 탭 → ＋
  2. 추가할 항목을 순서대로 선택 (메모 / 클립보드 항목 / 템플릿)
  3. 각 항목 사이의 대기 시간(초) 설정
  4. 저장 후 키보드의 콤보 탭에서 실행
대기 시간을 0.3초 이하로 하면 시스템이 못 따라올 수 있습니다. 0.5~1초를 권장합니다.
STEP 06

macOS 메뉴바 앱

Mac에서도 같은 메모·클립보드·콤보를 쓸 수 있습니다 (Mac Catalyst).

설치

App Store의 같은 앱에 포함됩니다. Mac App Store에서 ClipKeyboard를 검색해 설치하세요. iOS와 동일한 Apple ID로 로그인하면 iCloud로 데이터가 자동 동기화됩니다.

Mac 전용 기능

  • 메뉴바 아이콘에서 클릭 한 번으로 패널 열기
  • 전역 단축키 ⌃+⇧+V 등으로 바로 호출
  • 클립보드 모니터링 — 복사 즉시 스마트 분류
  • 부동 패널(Floating Panel)로 다른 앱 위에 띄워두기
Mac에서 자주 쓰는 문구를 저장해두면 iPhone 키보드에서도 바로 보이고, 그 반대도 마찬가지입니다.
STEP 07

iCloud 백업 & 복구

기기 변경·앱 재설치에도 메모를 잃지 않도록 iCloud에 백업할 수 있습니다.

백업하기

  1. 앱 설정 → iCloud 백업
  2. 지금 백업 탭 — 수동 백업
  3. 자동 백업 토글 — 5분 간격으로 변경 시 자동 저장

복구하기

  1. 새 기기에 앱 설치 후 같은 Apple ID로 로그인
  2. 설정 → iCloud 백업 → 복구
복구를 실행하면 현재 기기의 모든 메모·클립보드·콤보가 백업 데이터로 교체됩니다. 이 작업은 되돌릴 수 없습니다.
백업 용량은 무제한. 이미지 메모도 함께 저장됩니다.
FAQ

자주 묻는 질문 & 문제 해결

키보드가 추가한 메모를 못 보여요

"전체 접근 허용"이 꺼져 있을 가능성이 큽니다. iOS 설정 → 일반 → 키보드 → 클립키보드 → 전체 접근 허용을 확인하세요.

복사한 텍스트가 히스토리에 안 잡혀요

iOS는 기본적으로 클립보드를 다른 앱이 자동으로 읽는 것을 제한합니다. 앱을 한 번 켰다가 키보드를 사용하면 동기화됩니다.

IBAN을 복사했는데 카드번호로 인식돼요

숫자 길이·패턴이 유사하면 오분류가 있을 수 있습니다. 해당 항목을 길게 눌러 타입을 수동으로 바꾸거나, 공백·하이픈을 넣어서 다시 복사해보세요 (예: 1234-5678-...).

템플릿 변수에 넣은 값이 자동 저장되는데, 삭제는 어떻게 하나요?

템플릿 편집 화면에서 플레이스홀더를 탭하면 "저장된 값" 목록이 나옵니다. 스와이프로 개별 삭제할 수 있습니다.

콤보가 중간에 끊겨요 / 일부만 입력돼요

각 단계 사이 대기 시간을 늘려보세요 (0.5~1초 권장). 앱마다 입력 속도 허용치가 달라 빠르게 실행하면 일부 문자가 드롭될 수 있습니다.

Mac과 iPhone의 데이터가 동기화 안 돼요

두 기기가 같은 Apple ID로 로그인돼 있는지, iCloud Drive가 켜져 있는지 확인하세요. 앱 내 iCloud 백업 → 지금 백업을 한 번 눌러 수동 동기화도 가능합니다.

데이터는 어디에 저장되나요?

모든 메모는 당신의 기기(App Group 컨테이너)에만 저장됩니다. iCloud 백업을 활성화한 경우 암호화된 상태로 당신의 iCloud에 저장됩니다. 개발자는 어떤 데이터도 수집하지 않습니다.
