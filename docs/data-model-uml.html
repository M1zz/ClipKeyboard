동기화 / 마이그레이션

  • CloudKitBackupService — [Memo] · [SmartClipboardHistory] · [Combo] 를 CKAsset(JSON)으로 백업/복구. 자동 백업 5분 간격.
  • OldMemo → Memo — Memo(from:) 변환 생성자. 누락 필드는 기본값으로 채움.
  • ClipboardHistory → SmartClipboardHistory — 단순 히스토리에서 자동 분류 히스토리로 승격.
  • imageFileName → imageFileNames — 단일→다중 이미지. 둘 다 디코딩 지원(하위 호환).
  • 분류 우선순위 — 구체적 패턴 먼저: RRN → 사업자번호 → 카드번호 → 생년월일 → 계좌번호.

참고: iOS/macOS 공유본 Shared/Models/SharedModels.swift 가 동일 구조를 갖고, Model/Memo.swift 가 최신 정본입니다.
