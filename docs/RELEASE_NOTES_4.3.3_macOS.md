ClipKeyboard for Mac v4.3.3

한국어

- 아이폰에서 만든 메모가 맥에도 나타납니다 (베타), 같은 iCloud 계정이라면 아이폰과 맥 사이에서 메모가 자동으로 맞춰집니다 — 한쪽에서 추가하거나 고치거나 지운 내용이 잠시 뒤 다른 쪽에도 반영돼요. 보안 메모는 암호화된 채로 오갑니다 (실험적 기능이라, 아이폰 설정 → 기기 간 동기화에서 켜면 맥에서도 함께 동작합니다)
- 복원하면 사진까지 돌아옵니다, iCloud 백업에서 메모를 가져올 때 첨부 이미지도 함께 복원됩니다 — 맥을 새로 설치하고 아이폰 백업을 불러와도 사진 메모가 비어 있지 않아요. 맥은 처음 실행 시 데이터가 비어 있으면 백업을 자동으로 가져오는데, 이제 그 과정에 사진까지 포함됩니다
- 더 단단해진 속, 맥앱 전반의 글귀·아이콘·설정 키 관리를 아이폰 앱과 같은 단일 출처로 맞추고 메모 변경 알림 체계를 정리해, 동기화와 화면 갱신이 더 잘 맞물리도록 다듬었습니다

English

- Memos you make on iPhone now appear on your Mac (Beta), on the same iCloud account, memos keep themselves in step between iPhone and Mac — add, edit, or delete on one side and it shows up on the other a moment later. Secure memos travel encrypted (experimental — turn it on in iPhone Settings → Cross-Device Sync, and the Mac joins in automatically)
- Restores bring your photos back too, when you pull memos from an iCloud backup, their attached images come along — set up a fresh Mac and import an iPhone backup, and photo memos are no longer empty. The Mac already restores automatically when it starts up empty, and now that pass includes the pictures
- Sturdier under the hood, the Mac app's strings, icons, and settings keys now share a single source with the iPhone app, and the memo-change notification path was tidied so syncing and on-screen updates line up more reliably

배포용 요약 (한국어)

• 아이폰 ↔ 맥 메모 동기화(베타) — 같은 iCloud 계정에서 자동으로 맞춤, 보안 메모는 암호화 유지 (아이폰에서 켜면 맥도 함께 동작)
• 백업 복원 시 첨부 이미지 포함 — 새 맥에서 아이폰 백업을 불러와도 사진 메모가 그대로
• 시작 시 자동 복원(로컬이 비었을 때)도 이제 이미지까지 가져옴
• 내부 정리로 동기화·갱신 안정성 향상

Release Summary (English)

• iPhone ↔ Mac memo sync (Beta) — automatic on the same iCloud account, secure memos stay encrypted (enable on iPhone and the Mac joins in)
• Backup restore now includes attached images — import an iPhone backup on a new Mac and photo memos come back intact
• Startup auto-restore (when local is empty) now brings images too
• Internal cleanup for more reliable syncing and refreshes
