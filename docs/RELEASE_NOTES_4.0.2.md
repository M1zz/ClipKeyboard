# Release Notes v4.0.2

## 한국어 (Korean)

### 버전 4.0.2 업데이트

원격 프리랜서와 디지털 노마드를 위해 처음부터 다시 설계했어요. ✨

#### ✨ 새로운 기능

**💡 활용 사례 가이드 (앱 내)**
- 설정 → 활용 사례 메뉴에서 19가지 실전 시나리오 모음을 확인할 수 있어요
- 6개 카테고리: 결제·금융 / 타임존 / 프로페셔널 영어 / 플랫폼별 단축 / 노마드 라이프 / 스마트 클립보드
- IBAN 공유 · 타임존 응대 · 제안 후속 · 입금 독촉 · 스코프 푸쉬백 등 바로 쓰는 예시
- 각 시나리오 탭 한 번으로 **"메모로 저장"** 또는 **"복사"**

**📋 상단 캡처 카드**
- 방금 복사한 텍스트를 **자동 감지해서 상단에 카드로 표시**
- IBAN·이메일·URL 등 타입까지 자동 분류된 상태로 "메모로 저장" 버튼 제공
- 숨기기 한 번 더 뜨지 않음 (같은 값은 재노출 안 함)

**☀️ 시간대 인사말**
- 앱을 열 때마다 아침·오후·저녁·밤에 맞는 이모지와 인사로 시작
- 오늘 몇 번 사용했는지, 방금 전 무엇을 꺼냈는지 은근히 표시

#### 🎨 감성 리디자인

**심리스 스크롤**
- 섹션 헤더 대신 날짜 경계에만 **초미니멀 divider** (TODAY · YESTERDAY · EARLIER)
- 오래된 메모는 은은히 흐려져 뒤로 물러남 (recency fade)
- 앱 열 때 메모가 **차례로 부드럽게 나타나는** 스태거 애니메이션

**마이크로 인터랙션**
- 메모 탭 시 **햅틱 피드백** — 입력된 느낌 확실히
- 필터 전환 시 부드러운 crossfade
- 길게 누르면 **큰 미리보기 플로팅 팝업** (Mail 앱 스타일)
- 스크롤 시 상단 그리팅이 부드럽게 fade-out (Notes 앱 스타일)

#### 🖼 이미지 메모 개선

- 리스트에서 **실제 이미지 썸네일**이 아이콘 자리에 표시
- "이미지" 필터에 정확히 분류됨 (이전에 텍스트로 잡히던 문제 해결)

#### 🔐 개인정보 처리방침 & 웹 가이드

- 개인정보 처리방침 페이지 신설 — GDPR·CCPA 대응
- 웹 튜토리얼 페이지 오픈: `m1zz.github.io/ClipKeyboard/tutorial.html`

#### 🌍 통일성

- "Clip Keyboard" (띄어쓰기) → "ClipKeyboard" 일관 적용
- 한국어: "클립키보드" 통일 (기존 "클립 키보드" 혼용 제거)
- 앱 표시 이름 **언어별 자동 적용** (영어: ClipKeyboard / 한국어: 클립키보드)

## English

### Version 4.0.2 Update

Redesigned from the ground up for remote freelancers and digital nomads. ✨

#### ✨ What's New

**💡 Use Cases Guide (in-app)**
- 19 real-world scenarios under Settings → Use Cases
- 6 categories: Banking & Payments / Timezone / Professional English / Per-platform / Nomad Life / Smart Clipboard
- Examples: sharing IBAN, timezone replies, proposal follow-ups, payment reminders, scope pushback
- Tap any scenario to **Save as memo** or **Copy** instantly

**📋 Inline Capture Card**
- Automatically detects freshly copied text and shows a card at the top
- Already classified (IBAN, email, URL, etc.) with a "Save as memo" button
- Dismiss once — the same value won't reappear

**☀️ Time-aware greeting**
- Greeting and emoji that match your time of day (morning / afternoon / evening / night)
- Subtle stats: how many times you've tapped today, what you just used

#### 🎨 Seamless Redesign

**Continuous scroll, emotional feel**
- Section headers replaced with ultra-minimal day dividers (TODAY · YESTERDAY · EARLIER)
- Older memos gently fade into the background (recency fade)
- Items slide in smoothly when the app opens (stagger animation)

**Micro-interactions**
- Haptic feedback on every memo tap — confirms the input
- Filter transitions now crossfade smoothly
- Long-press reveals a **large floating preview** (Mail-style)
- Greeting fades as you scroll (Notes-style)

#### 🖼 Image memo improvements

- Actual image thumbnail shown in the list (replacing the generic text icon)
- Correctly filtered under "Image" (previously miscategorized as Text)

#### 🔐 Privacy & Web Guide

- New Privacy Policy page — GDPR & CCPA compliant
- Web tutorial now live at: `m1zz.github.io/ClipKeyboard/tutorial.html`

#### 🌍 Consistency

- "Clip Keyboard" (with space) → "ClipKeyboard" everywhere
- Korean: unified to "클립키보드" (no more mixed spacing)
- App name **adapts per language** automatically (English: ClipKeyboard / Korean: 클립키보드)
