ClipKeyboard for Mac v4.3.4

한국어

- 맥에서 iCloud 백업이 안정적으로 됩니다, 맥에서 백업이 실패하던 문제를 바로잡았습니다 — 이제 "백업하기"를 누르면 메모가 iCloud에 제대로 올라가고, 성공하면 "메모 N개를 백업했습니다"처럼 무엇이 백업됐는지 분명히 알려줍니다. 올릴 게 없거나 기존 백업을 지키느라 건너뛴 경우도 숨기지 않고 안내해, 됐는지 안 됐는지 헷갈릴 일이 없어요
- 데이터를 파일로 직접 보관하세요, 백업 화면에 "파일로 내보내기"와 "파일에서 가져오기"가 생겼습니다 — 메모·클립보드·콤보는 물론 첨부 사진까지 파일 하나에 담겨, iCloud 상태와 상관없이 안전한 곳에 보관하고 언제든 되살릴 수 있어요. 가져오기는 지금 데이터를 지우지 않고 합칩니다(같은 항목은 최신본 유지)
- 저장이 더 단단해졌습니다, 메모를 저장하는 도중 앱이 갑자기 닫혀도 원본이 깨지지 않도록 저장 방식을 안전하게 바꿨습니다 — 데이터가 어중간하게 손상될 일이 없습니다

English

- Reliable iCloud backup on Mac, fixed the issue where backup could fail on Mac — tapping "Backup" now puts your memos on iCloud properly, and on success it tells you exactly what went up, like "Backed up N memos." It also clearly says when there's nothing to back up or when a backup was skipped to protect an existing one, so you're never left guessing whether it worked
- Keep your data as a file, the backup screen now has "Export to File" and "Import from File" — memos, clipboard, combos, and even attached photos all go into a single file, so you can keep your data safe regardless of iCloud and bring it back anytime. Importing merges into your current data without erasing it (newest version wins for matching items)
- Sturdier saving, saving a memo now won't corrupt the original even if the app closes mid-save — no more half-written data

배포용 요약 (한국어)

• 맥 iCloud 백업 실패 문제 해결 — 백업이 제대로 올라가고, "메모 N개 백업됨"처럼 결과를 정확히 안내
• 파일로 내보내기·가져오기 추가 — iCloud 없이도 메모·사진까지 파일 한 개로 보관/복원 (가져오기는 합치기)
• 저장 안정성 강화 — 저장 도중 문제가 생겨도 원본 데이터가 손상되지 않음

Release Summary (English)

• Fixed Mac iCloud backup failures — backups go up properly with clear results like "Backed up N memos"
• New Export/Import to a file — keep or restore memos and photos in a single file without iCloud (import merges)
• Safer saving — your original data won't be corrupted if something interrupts a save
