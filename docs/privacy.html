개인정보 처리방침

최종 업데이트: 2026-04-25

🔒

ClipKeyboard는 어떤 개인정보도 수집·전송·공유하지 않습니다. 모든 데이터는 오직 당신의 기기에만 존재합니다.

1. 개요

ClipKeyboard("본 앱", "저희")는 사용자의 프라이버시를 최우선으로 합니다. 본 처리방침은 앱 사용 시 어떤 데이터가 어떻게 다뤄지는지 명시합니다.

핵심 원칙: 데이터 수집 없음(No Data Collection). 분석·트래킹·광고 SDK가 포함되지 않습니다.

2. 수집하지 않는 정보

본 앱은 다음 정보를 수집하지 않습니다:

  • 개인 식별 정보 (이름·이메일·전화번호·주소 등)
  • 사용자 행동 분석 (화면 이동·체류 시간·클릭 이벤트 등)
  • 기기 식별자 (IDFA·광고 ID·UUID 등)
  • 위치 정보
  • 저장된 메모·클립보드·템플릿 내용
  • 결제 정보 (Apple이 처리하며 개발자는 접근 불가)

3. 데이터 저장 위치

사용자가 앱에 입력한 모든 내용(메모·클립보드 히스토리·템플릿·콤보·설정 등)은 다음 위치에만 저장됩니다:

📱 기기 내부 (App Group 컨테이너)

iOS App Group 컨테이너 group.com.Ysoup.TokenMemo에 로컬 저장. 메인 앱과 키보드 익스텐션이 공유하지만, 개발자 또는 외부 서버로 전송되지 않습니다.

☁️ iCloud (선택)

사용자가 앱 내에서 iCloud 백업을 활성화한 경우에만 Apple의 CloudKit을 통해 사용자 본인의 iCloud 계정에 암호화된 상태로 저장됩니다. 개발자는 이 데이터에 접근할 수 없습니다.

iCloud 백업은 언제든지 비활성화할 수 있으며, Apple iCloud 정책에 따릅니다: Apple 개인정보 보호 정책

4. 권한 사용

본 앱이 요청하는 권한과 그 용도:

권한 용도 데이터 전송
전체 접근 허용 (키보드) 저장된 메모를 키보드에서 불러오기 위함 없음
Face ID / Touch ID 보안 메모 잠금 해제 (iOS가 처리) 없음
사진 라이브러리 이미지 메모 저장 (사용자가 선택한 사진만) 없음
iCloud 백업 활성화 시 사용자 iCloud 동기화 사용자 본인 iCloud로만

5. 써드파티 서비스

본 앱은 다음을 제외하고 어떤 써드파티 서비스도 포함하지 않습니다:

  • Apple CloudKit — iCloud 백업 기능에만 사용. 사용자가 활성화한 경우에만 작동
  • Apple StoreKit — 앱 내 구매 처리 (Apple이 처리, 개발자는 거래 내역만 확인)

분석 도구(Firebase·Mixpanel·Amplitude 등), 광고 네트워크, 크래시 리포팅 써드파티 서비스는 사용하지 않습니다.

6. 결제 처리

앱 내 구매(Pro unlock)는 Apple의 App Store가 처리합니다. 개발자는 다음만 확인합니다:

  • 구매가 성공했는지 여부 (영수증 검증)
  • 집계된 매출 데이터 (App Store Connect 제공)

카드 번호·청구 주소 등 결제 상세 정보는 개발자에게 전달되지 않습니다.

7. App Store Connect 분석

Apple이 자동으로 집계하여 개발자에게 제공하는 익명 통계 (다운로드 수·전환율·리텐션 등)를 확인합니다. 이 데이터는:

  • Apple이 익명화·집계한 상태로만 제공됨
  • 개별 사용자를 식별할 수 없음
  • 사용자가 iOS 설정에서 "앱 개발자와 공유" 토글을 켠 경우에만 포함됨

8. 사용자 권리 (GDPR / CCPA)

수집하는 데이터가 없기 때문에 대부분의 데이터 주체 권리는 자동으로 보장됩니다:

🇪🇺 GDPR (EU 사용자)

  • 접근권: 저장된 모든 데이터는 사용자 기기에 있으며 앱 내에서 직접 확인 가능
  • 삭제권: 앱을 삭제하거나 앱 내에서 데이터를 지우면 모든 정보가 영구 삭제됨
  • 이동권: iCloud 백업을 통해 다른 기기로 옮길 수 있음
  • 처리 제한권: 개발자가 데이터를 처리하지 않으므로 해당 없음

🇺🇸 CCPA (캘리포니아 사용자)

  • 개인정보를 수집·판매·공유하지 않습니다
  • "Do Not Sell" 옵션은 해당되지 않습니다 (판매 자체가 없음)

9. 어린이 개인정보 보호

본 앱은 만 13세 미만 아동을 특정하여 설계되지 않았으며, 해당 연령대로부터 의도적으로 정보를 수집하지 않습니다 (COPPA 준수).

10. 처리방침 변경
