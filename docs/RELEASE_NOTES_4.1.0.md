ClipKeyboard v4.1.0

한국어

- 메모 추가 화면을 전체적으로 새로 다듬었습니다, 백지에서 시작하는 부담을 줄이고 자연스러운 흐름으로 작성할 수 있도록 개선했습니다
- 카테고리(테마)를 선택하면 그 카테고리에 맞는 샘플 내용과 이름이 자동으로 채워집니다, 예를 들어 전화번호 카테고리는 010-0000-0000과 "전화번호" 이름이 미리 들어갑니다
- 샘플이 채워진 상태에서는 "샘플 — 수정해서 사용하세요" 안내가 노란색으로 표시되며, 한 글자라도 수정하면 자동으로 사라집니다
- 메모를 한 번도 추가하지 않은 분에게 페르소나에 맞는 예시 20여 종을 무작위로 3개씩 보여드립니다, 노마드는 비자 갱신·국제 인보이스·비동기 응답, 직장인은 회의록·이메일 서명, 학생은 교수 면담·과제 제출 등 — 진입할 때마다 새로운 영감
- "활용사례에서 영감 받기" 버튼을 추가했습니다, 실제 영어 비즈니스 템플릿을 카테고리 필터로 골라 메모로 가져올 수 있습니다
- 가져온 영어 템플릿의 [Your Name] 같은 더미 부분은 입력칸에서도 빨간 굵은 글씨로 강조되어 어디를 직접 수정해야 할지 한눈에 보입니다
- 시나리오를 메모로 저장할 때 그 시나리오가 템플릿이나 콤보 형식이면 메모 추가 화면의 해당 토글도 미리 켜져 있습니다, 사용자가 다시 판단할 필요 없이 바로 진행 가능
- 붙여넣을 내용 입력칸이 처음에는 작게 시작하고 내용이 많아지면 자동으로 늘어납니다, 최대 240pt 이후에는 내부 스크롤
- 키보드 위에 "다음" 버튼이 추가되어 붙여넣을 내용 → 키보드에 표시할 이름으로 즉시 이동할 수 있습니다
- 화면 순서를 카테고리 → 활용사례 → 붙여넣을 내용 → 키보드에 표시할 이름 → 옵션 순으로 재배치했습니다
- 라벨을 더 명확하게 변경했습니다, "제목" → "키보드에 표시할 이름", "내용" → "붙여넣을 내용"
- 메모 카드 배지를 시각적으로 구분했습니다, 일반 템플릿은 액센트색 TEMPLATE 배지, 옵션 연결 메모는 보라색 +TEMPLATE 배지
- 설정 → 키보드 → "페르소나"에서 온보딩 시 선택한 페르소나를 언제든 바꿀 수 있습니다, "사용 패턴"이라는 비슷한 이름과 겹치지 않도록 "페르소나"로 통일했습니다
- 활용사례 화면을 한국어와 인도네시아어로 완전히 번역했습니다, 영어 비즈니스 템플릿 본문은 사용자가 그대로 클라이언트에게 보낼 수 있도록 영어로 유지
- 클립보드 자동 분류의 우선순위를 개선했습니다, 계좌번호가 전화번호로 잘못 분류되던 문제를 수정하고 IBAN·SWIFT·VAT·카드번호·계좌번호 인식 정확도를 높였습니다
- 다국어 누락된 키를 일괄 보강했습니다, 영어와 인도네시아어 환경에서 한국어가 그대로 노출되던 문제를 해결

English

- The memo creation screen has been thoroughly reworked to reduce the "blank page" friction and guide you through a natural writing flow
- Picking a category now auto-fills both a sample value and a name for that category — e.g. picking Phone gives you 010-0000-0000 and the name "Phone" ready to edit
- When a sample is still in place, a yellow "Sample — please edit before using" notice appears, and disappears the moment you change a single character
- If you haven't added any memos yet, we now show 3 random examples from a pool of 20+ persona-tailored suggestions — nomad gets visa runs, international invoicing, and async-first replies; office workers get meeting minutes and email signatures; students get office hours and assignment submission. Fresh inspiration on every entry
- Added a "Get inspired by usage examples" button — browse real-world English business templates with a category filter and pull one into your memo
- In imported templates, dummy placeholders like [Your Name] are highlighted in bold red right inside the input field, so you instantly see what needs editing
- When you save a scenario as a memo, the corresponding toggle (Template / Combo) is now pre-enabled if the scenario calls for it — no need to decide again
- The "Content to paste" field starts compact and grows automatically as you type, with internal scrolling after ~240pt
- A "Next" button appears on the keyboard toolbar to jump from Content to Name in one tap
- Screen order rearranged: Category → Usage Examples → Content → Name → Options
- Labels are now meaningful: "Title" → "Name shown on keyboard", "Content" → "Content to paste"
- Memo card badges are visually distinct: regular TEMPLATE badge in accent color, attached-template memo gets a purple +TEMPLATE badge
- Settings → Keyboard → "Persona" lets you change the persona you picked during onboarding at any time, unified under the single "Persona" label to avoid confusion with the "Use case" entry
- The Usage Examples screen is fully translated into Korean and Indonesian — English business template bodies stay in English so you can send them to clients as-is
- Improved clipboard auto-classification priority — fixed bank account numbers being misclassified as phone numbers, with better detection accuracy for IBAN, SWIFT, VAT, credit card, and bank account
- Filled in missing translations across the board — no more Korean strings leaking into the English or Indonesian UI
