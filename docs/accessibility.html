시각·운동·청각 보조 기능을 완전히 지원합니다. Apple 손쉬운 사용 철학을 바탕으로, 어떤 방식으로 기기를 사용하든 ClipKeyboard를 자유롭게 쓸 수 있습니다. Full support for visual, motor, and auditory assistive features. Built on Apple's accessibility philosophy — use ClipKeyboard your way, no matter how you interact with your device.

지원하는 손쉬운 사용 기능 Supported Accessibility Features

아래 기능들은 iPhone 설정 → 손쉬운 사용에서 켤 수 있습니다. Enable these features in iPhone Settings → Accessibility.

👁

VoiceOver

메모 목록에서 한 번 탭하면 카테고리·제목·내용을 읽어줍니다. 이중 탭하면 클립보드에 복사되며 "복사됨" 알림이 즉시 읽힙니다.

위아래 쓸기로 즐겨찾기 추가, 삭제 커스텀 액션에 접근할 수 있어 별도 스와이프 없이 사용 가능합니다. One tap on a memo row announces category, title, content, and timestamp. Double-tap copies to clipboard and immediately announces "Copied".

Swipe up/down to access Add Favorite and Delete custom actions without requiring swipe gestures.

⚙️ 설정 → 손쉬운 사용 → VoiceOver ⚙️ Settings → Accessibility → VoiceOver

🎙

음성 명령 Voice Control

화면에 보이는 버튼 이름을 그대로 말해 조작합니다. "메모 검색 탭", "확인 탭"처럼 말하면 됩니다. 모든 버튼과 입력 필드에 읽기 가능한 이름이 지정되어 있습니다. Say the name of any visible button to activate it. "Tap Search", "Tap Done", and more. Every button and input field has an accessible name for precise voice targeting.

⚙️ 설정 → 손쉬운 사용 → 음성 명령 ⚙️ Settings → Accessibility → Voice Control

🔠

더 큰 텍스트 Larger Text

시스템 텍스트 크기를 키우면 메모 제목·내용 미리보기·배지가 비례해서 커집니다. 커스텀 폰트(Paper 테마)도 Dynamic Type 기반으로 스케일됩니다.

앱 내 글꼴 크기 슬라이더로 키보드만 별도 조정할 수도 있습니다. Increasing system text size proportionally scales memo titles, previews, and badges. Custom fonts (Paper theme) scale via Dynamic Type.

An in-app font size slider lets you adjust the keyboard separately.

⚙️ 설정 → 디스플레이 및 밝기 → 텍스트 크기 ⚙️ Settings → Display & Brightness → Text Size

🌙

다크 모드 Dark Mode

Dusk·Paper 두 테마 모두 다크 모드를 완전히 지원합니다. 시스템 설정을 따르거나 앱 내에서 직접 선택할 수 있습니다. 밝기에 민감한 사용자에게 권장됩니다. Both Dusk and Paper themes fully support dark mode. Follow the system setting or choose manually in-app. Recommended for users sensitive to bright screens.

⚙️ 설정 → 디스플레이 및 밝기 → 다크 ⚙️ Settings → Display & Brightness → Dark

🎨

색상 없이 구별 Differentiate Without Color

Template · +Template · Combo · New 배지는 색상뿐 아니라 텍스트 라벨로도 구분됩니다. 색맹 사용자도 배지 종류를 읽을 수 있습니다. Template, +Template, Combo, and New badges are distinguished by text labels, not color alone. Color-blind users can read badge types without relying on color.

⚙️ 설정 → 손쉬운 사용 → 디스플레이 및 텍스트 크기 → 색상 필터 ⚙️ Settings → Accessibility → Display & Text Size → Color Filters

🌀

동작 줄이기 Reduce Motion

즐겨찾기 추가 시 나타나는 하트 애니메이션이 동작 줄이기 설정을 자동으로 따릅니다. 켜져 있으면 이동·스케일 효과 없이 페이드만 사용합니다. 전정 장애가 있는 사용자에게 권장됩니다. The heart animation on favoriting automatically respects Reduce Motion. When enabled, only a fade is used — no movement or scale effects. Recommended for users with vestibular disorders.

⚙️ 설정 → 손쉬운 사용 → 동작 → 동작 줄이기 ⚙️ Settings → Accessibility → Motion → Reduce Motion

🔒

충분한 대비 Sufficient Contrast

Dusk·Paper 두 테마 모두 WCAG 2.1 AA 기준(4.5:1) 이상의 색상 대비를 제공합니다. 저시력 사용자도 본문 텍스트와 배경을 명확히 구별할 수 있습니다. Both Dusk and Paper themes meet WCAG 2.1 AA contrast requirements (4.5:1 minimum). Users with low vision can clearly distinguish body text from backgrounds.

⚙️ 설정 → 손쉬운 사용 → 디스플레이 및 텍스트 크기 → 대비 높이기 ⚙️ Settings → Accessibility → Display & Text Size → Increase Contrast

✅ 직접 테스트하는 방법 How to Test

  1. 1 VoiceOver — 메모 행 한 번 탭 → 전체 내용 읽기 확인. 이중 탭 → "복사됨" 알림 확인. 위아래 쓸기 → 즐겨찾기·삭제 액션 확인. VoiceOver — Single-tap a memo row, verify full content is read. Double-tap, verify "Copied" announcement. Swipe up/down, verify Favorite and Delete actions appear.
  2. 2 음성 명령 — "탭 검색" 또는 "탭 확인" 말하기 → 반응 확인. Voice Control — Say "Tap Search" or "Tap Done" and verify the app responds.
  3. 3 더 큰 텍스트 — 설정에서 텍스트 크기 최대로 → 메모 목록 텍스트 크기 확인. Larger Text — Set system text size to maximum. Verify memo titles and previews scale proportionally.
  4. 4 다크 모드 — 제어 센터에서 다크 모드 전환 → 즉시 색상 변경 확인. Dark Mode — Toggle dark mode from Control Center. Verify colors update instantly without relaunching.
  5. 5 색상 필터 — 설정 → 적록색맹 필터 ON → 배지 구분 확인. Color Filters — Enable red-green color blindness filter. Verify badges are distinguishable by text alone.
  6. 6 동작 줄이기 — 설정 ON → 즐겨찾기 추가 → 하트 애니메이션이 이동 없이 페이드만 되는지 확인. Reduce Motion — Enable setting → add a favorite → verify the heart only fades, no movement or scale.
