## 🅰️ 한국어 단일 포스트 (바로 복붙용)

```
원격으로 일해보면 알아요.
힘든 건 일이 아니라, 같은 2분짜리 잡일이 하루에 20번 반복되는 거였어요.

• 새 거래처마다 계좌번호 다시 타이핑 (오타 한 번 = "입금했는데요?" 핑퐁 사흘)
• 똑같은 정산·회신 문구 또 쓰기
• 내 근무 시간대 또 설명하기

하루 20분을 돌려준 방법:
반복되는 건 딱 한 번만 써두고, '메모앱'이 아니라 키보드에 넣어두고, 짧은 이름을 붙여요 — "계좌", "정산", "회신".
그러면 카톡·메일·슬랙 어디서든 탭 한 번.
```

## 🅱️ 한국어 스레드 버전 (3개 연속 — 도달·참여에 더 유리)

**1/3 (훅 + 아픔)**
```
원격으로 일하면서 깨달았어요.
힘든 건 일 자체가 아니라, 같은 2분짜리 잡일이 하루 20번 반복되는 거더라고요.

새 거래처마다 계좌번호 다시 타이핑하다가 오타 한 번 → "입금했는데 안 들어왔어요?" 핑퐁 사흘. 😮‍💨

하루 20분 돌려준 방법 정리해봤어요 🧵
```

**2/3 (방법 — 앱 없이도 가능)**
```
핵심은 "빨리 치자"가 아니라 "같은 걸 두 번 치지 말자"예요.

1. 반복되는 건 딱 한 번만 써두기 — 계좌 정보, 자주 보내는 답장 3개, 근무 시간대 한 줄.
2. '메모앱'이 아니라 키보드에 넣기. 진짜 마찰은 대화창을 나가서 복사하러 가는 그 동선이에요.
3. 짧은 이름 붙이기 — "계좌", "정산", "회신". 손이 기억하게.
```

## 운영 팁
- **첫 줄이 전부**: Threads는 첫 문장에서 스크롤을 못 멈추면 끝. 아픔을 1초 안에.
- **링크 위치 트레이드오프**: Meta는 본문에 외부 링크가 있으면 도달을 약간 깎는 경향이 있음.
  - 도달 우선이면 → 본문엔 링크 빼고, **첫 답글에 링크**(`?pt=...&ct=threads` 캠페인 파라미터 붙여서).
  - 전환 우선/팔로워 적을 때 → 위처럼 본문에 그냥 넣어도 됨.
- **해시태그**: 1개만 (`#프리랜서` 또는 `#원격근무`).
- **이미지/영상 1개**: 카톡·메일 안에서 키보드로 계좌번호를 탭하는 6초 영상이 텍스트보다 강함.
- **오퍼코드(선택)**: 현재 `2026JUNE` 리딤 링크가 있음 → 런칭 특가 줄 거면 마지막 줄에
  `런칭 특가: https://apps.apple.com/redeem?ctx=offercodes&id=1543660502&code=2026JUNE` (할인폭·만료일 확인 후 사용).
- **참여 유도**: 마지막 질문에 달리는 답글이 곧 다음 글감 + 도달.
