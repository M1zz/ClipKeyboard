# Release Notes v4.0.6

## 한국어 (Korean)

### 버전 4.0.6 업데이트

#### 무료 한도가 더 넉넉해졌어요

- **메모 5개 → 10개** — 처음 며칠 동안 충분히 가치를 체감하고 결정할 수 있도록 확장.
- **콤보 1개 → 3개** — 콤보 기능 자체를 제대로 학습할 수 있는 여유 확보.
- **클립보드 기록 20개 → 50개** — 노마드 작업 흐름에서 한두 시간 만에 잘리던 문제 해소.
- **이미지 메모 5개까지 무료** — 명함, 영수증, 여권 사진 같은 핵심 use case가 무료에서도 가능해졌어요.
- **테마 커스터마이징 무료 개방** — 키보드 색상은 이제 모든 사용자에게.

기존 v4.0 사용자는 그랜드파더 처리되어 새 한도와 무관하게 기존처럼 사용하실 수 있어요.

#### 7일 무료 체험 신설

- Pro 업그레이드 화면에 "**7일 무료 체험 시작**" 버튼이 추가됐어요. 한 번만 눌러 7일 동안 모든 Pro 기능(무제한 메모, iCloud 백업, 생체인증 잠금, 무제한 이미지 메모 등)을 직접 써보실 수 있어요.
- 체험 기간 중에는 설정 화면에 "체험 활성 — N일 남음" 표시.
- 체험은 1회만 가능, 만료되면 자동으로 무료 한도로 돌아갑니다.

#### 키보드 X(전체 지우기) 버튼 표시 버그 수정

- 메모를 키보드로 입력한 직후 X 버튼이 보이지 않던 문제를 수정했어요. 이제 입력하자마자 X 버튼으로 깔끔히 지울 수 있어요.
- 호스트 앱 텍스트 필드에 이미 텍스트가 있는 상태로 키보드를 열어도 X 버튼이 즉시 표시됩니다.

#### 사용 통계 정확화 + 시간 절약 표시

- "오늘 N번 사용" 카운트가 이전엔 매일 정확히 초기화되지 않아 누적값처럼 보이던 문제 수정. 이제 자정에 자연스럽게 0부터 시작합니다.
- 메인 화면 그리팅 아래에 평생 절약한 시간이 작은 배지로 노출돼요 (예: "Saved 4h 32m so far"). 누적 10분 이상부터 표시.

#### 결제 진입 경로 개선

- 키보드 사용 중 메모 한도에 가까워지면 (남은 슬롯 2개 이하) 미리 안내 배너가 떠요. "메모 한도까지 N개 남음" 형태로 미리 알 수 있어요.

## English

### Version 4.0.6 Update

#### More generous free tier

- **Memos: 5 → 10** — enough room to actually feel the value before deciding.
- **Combos: 1 → 3** — finally enough to learn what combos are good for.
- **Clipboard history: 20 → 50** — no more silent truncation after an hour or two of nomad work.
- **Image memos: 5 free** — business cards, receipts, passport photos and other essential use cases are now usable on the free tier.
- **Theme customization is now free** — keyboard colors are open to everyone.

Existing v4.0 users are grandfathered and unaffected by the new limits.

#### New: 7-day free trial

- The Pro upgrade screen now offers a "**Start 7-day free trial**" button. One tap unlocks all Pro features (unlimited memos, iCloud backup, biometric lock, unlimited image memos, etc.) for 7 days so you can try them yourself.
- During the trial, Settings shows a "Trial active — N days left" row.
- The trial is one-time only and falls back to the free tier automatically when it ends.

#### Keyboard clear-all (X) button bug fix

- Fixed a bug where the X (clear-all) button wouldn't appear right after inserting a memo via the keyboard. You can now clear what you just typed instantly.
- The X button also shows up immediately when the keyboard opens on a text field that already contains text.

#### More accurate usage stats + time-saved badge

- The "N taps today" counter wasn't really resetting daily — it leaked lifetime totals into today's number. Now it cleanly starts from zero at local midnight.
- A small badge under the home screen greeting shows your lifetime time saved (e.g. "Saved 4h 32m so far"). Appears once cumulative savings cross 10 minutes.

#### Smoother path to upgrade

- A heads-up banner appears in the keyboard when you get close to the memo limit (2 or fewer slots left), so you're never surprised when you hit it.
