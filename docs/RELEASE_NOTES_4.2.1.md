ClipKeyboard v4.2.1

한국어

- Pro 활성화 표시를 바로잡았습니다, 예전 유료 버전을 구매했던 분들과 TestFlight 사용자에게 모든 기능이 이미 열려 있는데도 설정 화면에 "Pro 업그레이드" 안내가 계속 뜨던 문제를 해결했습니다 — 이제 "Pro 활성화됨"으로 정확히 표시됩니다
- 카테고리를 처음부터 비어 있게 바꿨습니다, 미리 채워져 있던 십수 개의 기본 카테고리를 없앴습니다 — 전체·즐겨찾기 페이지는 항상 제공되고 나머지는 필요한 것만 직접 만들어 쓰면 됩니다, 목록이 한결 깔끔해집니다
- 카테고리 자동 제안을 추가했습니다, 같은 종류(이메일·전화번호 등)로 분류된 메모가 쌓이면 "이 카테고리를 만들어 한 곳에 모아드릴까요?"라고 부드럽게 안내합니다 — 한 번 탭하면 카테고리가 생기고 그 메모들이 바로 그 페이지에 모입니다
- 메모를 길게 눌러 카테고리를 옮길 수 있습니다, 메모 액션 시트에서 다른 카테고리로 이동하거나 새 카테고리를 즉석에서 만들어 바로 담을 수 있습니다
- "테마 선택" 용어를 정리했습니다, 메모를 만들 때 보이던 "테마 선택"을 없애 "카테고리"로 통일했습니다 — 더 이상 테마와 카테고리가 헷갈리지 않으며, 카테고리 관리 화면과 키보드 페이지가 완전히 같은 목록을 사용합니다
- 키보드 메모 버튼이 카테고리 색으로 은은하게 구분됩니다, 동시에 카테고리가 없는데도 메모에 색이 칠해지던 문제를 해결했습니다 — 직접 만든 카테고리에 속한 메모에만 색이 들어갑니다
- 즐겨찾기 색을 더 선명한 분홍으로 통일했습니다, 탭·하트 배지·키보드 즐겨찾기 페이지가 한 가지 지정색으로 맞춰집니다
- 여러 화면의 영어 번역을 보강했습니다, 영어로 표시되지 않고 한국어가 그대로 노출되던 문구들을 번역으로 채웠습니다

English

- Fixed the Pro status display, long-time paid users and TestFlight testers had every feature unlocked but still saw an "Upgrade to Pro" prompt in Settings — it now correctly shows "Pro activated"
- Categories now start empty, removed the dozen-plus pre-seeded default categories — the All and Favorites pages are always there, and you create just the categories you actually need, so the list stays clean
- Added smart category suggestions, when several memos of the same kind (email, phone, etc.) pile up, a gentle tip offers to "Create this category and gather them in one place" — one tap creates the category and the matching memos appear there instantly
- Move memos to a category with a long-press, the memo action sheet now lets you move a memo to another category, or create a new category on the spot and drop the memo straight into it
- Cleaned up the "Theme" wording, the "Theme" picker that appeared while creating a memo is gone and everything is now called "Category" — no more confusion between themes and categories, and the category management screen and keyboard pages share the exact same list
- Keyboard memo buttons are now subtly tinted by their category color, and we fixed a bug where memos were colored even when you had no categories — only memos in a category you created get a tint now
- Unified the Favorites color to a brighter pink, the tab, heart badges, and the keyboard favorites page now use one designated pink
- Improved English translations, filled in strings across several screens that previously showed Korean text instead of English
